# Contributing to Frequent Traders 🤝

Thank you for your interest in contributing! Here's how you can help:

## 🐛 Reporting Bugs

Before creating a bug report:
1. Check existing [issues](https://github.com/mermotec/Frequent-Traders/issues)
2. Make sure you're using the latest version
3. Test without other mods to isolate the issue

### Bug Report Should Include:
- **Minecraft version** (e.g., 1.21.1)
- **Mod loader** (Forge/NeoForge/Fabric) and version
- **Mod version** of Frequent Traders
- **Description** of the bug
- **Steps to reproduce**
- **Expected behavior**
- **Screenshots** (if applicable)
- **Crash log** or latest.log (use pastebin/gist)

## 💡 Suggesting Features

Feature requests are welcome! Please include:
- Clear description of the feature
- Why it would be useful
- How it should work
- Mockups or examples (if applicable)

## 🛠️ Pull Requests

### Before Submitting:
1. Fork the repository
2. Create a new branch (`feature/your-feature-name`)
3. Test your changes thoroughly
4. Follow existing code style
5. Update documentation if needed

### PR Guidelines:
- Clear title describing the change
- Reference related issues (e.g., "Fixes #123")
- Describe what changed and why
- Include screenshots for UI changes

### Code Style:
- Use descriptive variable names
- Comment complex logic
- Follow Java naming conventions
- Keep methods focused and concise

## 📖 Documentation

Help improve documentation by:
- Fixing typos
- Clarifying instructions
- Adding examples
- Translating to other languages

## ❓ Questions?

Not sure about something? Open a [Discussion](https://github.com/mermotec/Frequent-Traders/discussions) or create an issue!

## 🚀 Development Setup

### Requirements:
- JDK 21 or newer
- IntelliJ IDEA (recommended) or Eclipse
- Gradle (included via wrapper)

### Setup:
```bash
git clone https://github.com/mermotec/Frequent-Traders.git
cd Frequent-Traders
./gradlew build
```

### Running in Development:
```bash
./gradlew runClient  # Test client
./gradlew runServer  # Test server
```
