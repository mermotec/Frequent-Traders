## ✨ Features
- 🔄 Greatly increased Wandering Trader spawn rate
- ⚙️ Fully configurable timing & chance via `config/`
- 🦙 Optional llamas toggle
- 🧩 Compatible with most mods

## 📥 Install
1. Install [Forge](https://files.minecraftforge.net/) **or** [NeoForge](https://neoforged.net/) for your MC version
2. Drop the `.jar` into your `mods/` folder
3. Launch and enjoy traders everywhere!

> ⚠️ **Modrinth mirror is outdated** — always download the latest from [CurseForge](https://www.curseforge.com/minecraft/mc-mods/frequent-traders).
