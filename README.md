A Minecraft mod that **greatly increases the spawn rate of Wandering Traders** with a flexible and easy-to-configure system.

## 👋 About

Tired of waiting hours for Wandering Traders to spawn? **Frequent Traders** solves this problem by significantly increasing their spawn frequency, making it easier to access their unique trades and items. Perfect for peaceful mode players, skyblock challenges, or anyone who wants more frequent trading opportunities!

## ✨ Features

- **Increased Spawn Rates**: Wandering Traders spawn much more frequently than vanilla
- **Configurable System**: Adjust spawn rates to your preference
- **Lightweight**: Minimal performance impact
- **Compatible**: Works with most other mods
- **Easy to Use**: No complex setup required

## 🛠️ Installation

1. Download and install [Forge](https://files.minecraftforge.net/) / [NeoForge](https://neoforged.net/) / [Fabric](https://fabricmc.net/)
2. Download the latest version of Frequent Traders from CurseForge
3. Place the `.jar` file in your `mods` folder
4. Launch Minecraft and enjoy more frequent traders!

## ⚙️ Configuration

The mod includes a configuration file that allows you to customize:
- Spawn frequency multiplier
- Minimum/maximum spawn intervals
- Enable/disable features

Configuration file location: `config/frequent-traders.toml` (or similar)
